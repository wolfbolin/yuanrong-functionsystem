.. _get_instance:

yr.InstanceCreator.get_instance
----------------------------------------------------------------

.. py:method:: InstanceCreator.get_instance(name, *args, **kwargs)

    在集群中获取一个实例。

    参数:
        - **name** (str) - 用户类。
        - **\*args** - 可变参数，用于传递非关键字参数。
        - **\*\*kwargs** - 可变参数，用于传递关键字参数。

    返回：
        InstanceProxy。
