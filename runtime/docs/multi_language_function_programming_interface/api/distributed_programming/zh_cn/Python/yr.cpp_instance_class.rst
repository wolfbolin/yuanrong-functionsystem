yr.cpp_instance_class
=======================

.. py:class:: yr.cpp_instance_class(class_name: str, factory_name: str, function_urn: str)

    基类：``object``

    为 cpp 类构造代理以启用远程调用。

    **方法**：

    .. list-table::
       :header-rows: 0
       :widths: 30 70

       * - :ref:`__init__ <init_cpp>`
         - 用于构造 cpp 类的代理，远程调用 cpp 类。
       * - :ref:`get_class_name <get_class_name>`
         - 获取此 CPP 实例中的类名。
       * - :ref:`get_factory_name <get_factory_name>`
         - 获取此 CPP 实例中的工厂函数名。
       * - :ref:`get_function_key <get_function_key>`
         - 获取此 CPP 实例中的功能键。
       * - :ref:`invoke <invoke>`
         - 创建一个 cpp class 的实例。
       * - :ref:`options <options>`
         - 设置用户调用选项。
